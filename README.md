# webbed-site

Personal portfolio / CV site built with Angular 21, Tailwind CSS 4, and Phosphor Icons.

## Commit history

Squashed into a single commit on purpose.

## Getting started

```bash
npm install
npm start
```

Dev server runs at `http://localhost:4200`.

## Build

```bash
npm run build
```

Output goes to `dist/webbed-site`.
